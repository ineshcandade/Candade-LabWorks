# Comparative Analysis of Free-Space and Metal-Mounted Yagi Antenna

---

This project presents a MATLAB-based simulation and optimization of a **Yagi-Uda antenna operating at 1.9 GHz** using the MATLAB Antenna Toolbox.

The simulation investigates the performance of the antenna under two different environments: **free-space** and **metal-mounted conditions**. It analyzes how nearby metallic structures influence antenna characteristics and demonstrates how re-optimization can restore performance.

These simulations provide practical insight into antenna design, electromagnetic wave propagation, impedance matching, and radiation pattern analysis using computational modeling.

---

## What It Does

- Designs a Yagi-Uda antenna for 1.9 GHz operation
- Optimizes director lengths and element spacing
- Compares free-space and metal-mounted antenna performance
- Evaluates gain, VSWR, impedance, and front-to-back ratio
- Generates 3D radiation patterns, E-plane, and H-plane plots
- Estimates antenna bandwidth using VSWR vs Frequency

## How It Works

The MATLAB program begins by designing a Yagi-Uda antenna using standard design equations for a target frequency of **1.9 GHz**.

For antenna optimization:
- Director lengths and element spacing are automatically varied
- Multiple antenna configurations are evaluated
- The best design is selected based on gain, front-to-back ratio, and VSWR

For metal-mounted analysis:
- A conductive reflector is introduced to simulate metal mounting
- Antenna performance is evaluated at different mounting heights
- The antenna is re-optimized to compensate for performance degradation

For visualization:
- 3D radiation patterns are generated
- E-plane and H-plane radiation patterns are compared
- VSWR is evaluated across the operating frequency range to estimate bandwidth
