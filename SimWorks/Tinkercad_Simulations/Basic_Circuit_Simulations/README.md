# Basic Circuit Simulations 

---

This project contains a collection of fundamental Arduino-based circuit simulations developed using Tinkercad Circuits.  
The simulations explore basic embedded and electronics concepts through simple interactive examples built in a virtual environment.

These experiments help visualize circuit behavior and microcontroller interactions before implementing similar systems on real hardware.

---

## Simulations Included
- LCD Welcome Screen
- LED Blinking Circuit
- Freeze-Frame LED Toggle 
- Basic Traffic System
- Interactive Obstacle Detector

---

## Platform
- Tinkercad Circuits
- Arduino (Simulated)

---

## Purpose
The goal of these simulations is to experiment with basic circuit design, understand microcontroller I/O behavior, and build foundational skills through safe virtual prototyping.
